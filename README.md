# AI Engineer Journey

A structured, project-based program building toward **production AI engineering** — from
strong software foundations through machine learning, deep learning, and the modern
LLM / RAG / agent stack, with an emphasis on **deployed, real-world projects** rather than
theory alone.

This repository is my public, day-by-day learning log. Each commit is a concept learned
and applied by hand, and the program culminates in a set of deployed portfolio projects.

---

## Background

Coming into this with a foundation in:

- **Python**, data handling (NumPy, Pandas), and SQL
- **Embedded systems & IoT** (ESP32, Arduino) from an Electronics & Communication
  Engineering background
- **Computer vision** exposure from an AR-based indoor-navigation thesis

This program deepens that foundation specifically toward applied / GenAI engineering.
